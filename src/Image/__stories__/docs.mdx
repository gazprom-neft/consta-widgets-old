# Image

Изображение. Используется в более сложных компонентах. Задается по ссылке в свойстве `src`.

<ImageExample />

## Свойства

<!-- props:start -->

| Свойство     | Тип               | По умолчанию | Описание                 |
| ------------ | ----------------- | ------------ | ------------------------ |
| `src`        | `string`          | -            | Ссылка на изображение    |
| `className?` | `string`          | -            | Дополнительный CSS-класс |
| `title?`     | `React.ReactNode` | -            | Заголовок                |

## Пример

```
<Image src="https://images.pexels.com/photos/414612/pexels-photo-414612.jpeg" />
```
