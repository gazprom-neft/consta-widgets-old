# ImagesList

Карусель изображений. Показывает миниатюры изображений в ряд.

- [Ссылки на изображения](#ссылки)
- [Активный элемент](#активный-элемент)
- [Список свойств](#свойства)
- [Пример использования](#пример)

## Ссылки

Ссылки на изображения задаются в объекте `image`:

- в `large` указывается ссылка на основное изображение,
- в `preview` (необязательно) можно дать ссылку на миниатюру. Если не укажете `preview`, миниатюра
  сформируется автоматически из изображения в `large`.

<ImagesListExample />

```tsx
<ImagesList
  images={[
    { large: 'https://images.pexels.com/photos/414612/pexels-photo-414612.jpeg' },
    { preview: 'https://via.placeholder.com/150', large: 'https://via.placeholder.com/1500' },
    { large: 'https://via.placeholder.com/400x200' },
    { large: 'https://via.placeholder.com/200x400' },
  ]}
  onClick={action('onClick')}
/>
```

## Активный элемент

Вы можете выбрать, какое изображение по порядку будет активным.
Задайте его номер в свойстве `activeItem`.

<ImagesListExampleActive />

```tsx
<ImagesList
  images={[
    { large: 'https://images.pexels.com/photos/414612/pexels-photo-414612.jpeg' },
    { preview: 'https://via.placeholder.com/150', large: 'https://via.placeholder.com/1500' },
    { large: 'https://via.placeholder.com/400x200' },
    { large: 'https://via.placeholder.com/200x400' },
  ]}
  onClick={action('onClick')}
  activeItem={0}
/>
```

## Свойства

<!-- props:start -->

| Свойство                           | Тип                     | По умолчанию | Описание                                    |
| ---------------------------------- | ----------------------- | ------------ | ------------------------------------------- |
| [`images`](#ссылки)                | `ImageItem[]`           | -            | Объект со списком изображений               |
| [`activeItem?`](#активный-элемент) | `number`                | -            | Номер изображения, которое нужно подсветить |
| `onClick`                          | `(idx: number) => void` | -            | Действие по клику на изображение            |
| `className?`                       | `string`                | -            | Дополнительный CSS-класс                    |
