# ImagesPopup

Галерея изображений. Состоит из большого изображения и карусели из миниатюр, с помощью
которой можно выбрать, какое изображение увеличить.

## Ссылки

Ссылки на изображения задаются в объекте `image`:

- в `large` указывается ссылка на основное изображение,
- в `preview`(необязательно) можно дать ссылку на миниатюру. Если не укажете `preview`, миниатюра
  сформируется автоматически из изображения в `large`.

## Развёрнутое изображение

Вы можете выбрать, какое изображение будет развёрнуто при загрузке галереи.
Задайте его номер по порядку в свойстве `openOnImage`.

## Свойства

<!-- props:start -->

| Свойство         | Тип           | По умолчанию | Описание                                              |
| ---------------- | ------------- | ------------ | ----------------------------------------------------- |
| `images`         | `ImageItem[]` | -            | Объект со списком изображений                         |
| `openOnImage?`   | `number`      | -            | Номер изображения, которое нужно развернуть на старте |
| `onRequestClose` | `() => void`  | -            | Действие по закрытию изображения                      |
| `className`      | `string`      | -            | Дополнительный CSS-класс                              |

## Пример

```
<ImagesPopup
  images={[
    { large: 'https://images.pexels.com/photos/414612/pexels-photo-414612.jpeg' },
    { preview: 'https://via.placeholder.com/150', large: 'https://via.placeholder.com/1500' },
    { large: 'https://via.placeholder.com/400x200' },
    { large: 'https://via.placeholder.com/200x400' },
  ]}
  openOnImage={1}
  onRequestClose={action('onRequestClose')}
/>
```
