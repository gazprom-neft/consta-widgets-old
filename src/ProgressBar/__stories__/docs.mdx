## Данные

Данные для диаграммы описываются в объекте `data`.

### Цвет и значения

За цвет отвечает свойство `color`.

Начальное и конечное значение (план) указываются в свойствах `valueMin` и `valueMax`.

Длина цветной линии (то, чего достигли в реальности) — в свойстве `value`.

Подпись к линии (показывается рядом с ней, число или строка) добавляется в свойстве `summary`.

<ProgressBarExample />

```tsx
<ProgressBar
  data={
    [
      {
        value: 50,
        valueMin: 0,
        valueMax: 100,
        summary: '50 штук',
        color: 'var(--color-bg-success)'
      },
    ],
  }
  size="s"
/>
```

### Заголовок

В `caption` можно добавить заголовок.
Если линий на диаграмме несколько, заголовки лучше согласовать между собой.

<ProgressBarExampleCaption />

### Без данных

На диаграмме можно показать, что данных нет: работа по проекту не началась или просто ничего не известно.

<ProgressBarExampleNull />

## Деления

На диаграмме можно показать деления, они задаются в массиве `ticks`. Внутри:

- `value` — значение, у которого нужно поставить деление,
- `label` — подпись к делению (число или строка).

<ProgressBarExampleTicks />

```ts
// ticks
[
  { "value": 0, "label": 0 },
  { "value": 25, "label": "" },
  { "value": 50, "label": 50 },
  { "value": 75, "label": "" },
  { "value": 100, "label": 100 }
]
```

## Размер диаграммы

За размер компонента отвечает свойство `size`. Варианты: `'xs'`, `'s'`, `'m'`, `'l'` .

Меняется толщина линий на диаграмме, размер шрифта и отступы.
Максимальная ширина диаграммы равна ширине родительского контейнера.
Минимальная ширина для диаграммы размера `'xs'` — 100px, для остальных размеров — 160px.

## Свойства

<!-- props:start -->

| Свойство                     | Тип                         | По умолчанию | Описание                       |
| ---------------------------- | --------------------------- | ------------ | ------------------------------ |
| [`data`](#данные)            | [`Data[]`](#data)           | -            | Массив с данными для диаграммы |
| [`size?`](#размер-диаграммы) | `'xs'`, `'s'`, `'m'`, `'l'` | -            | Размер                         |
| `title?`                     | `React.ReactNode`           | -            | Заголовок                      |

### Data

| Свойство                       | Тип               | По умолчанию | Описание                     |
| ------------------------------ | ----------------- | ------------ | ---------------------------- |
| [`value`](#цвет-и-значения)    | `string`          | -            | Основное значение — прогресс |
| [`valueMin`](#цвет-и-значения) | `string`          | -            | Начальное значение           |
| [`valueMax`](#цвет-и-значения) | `string`          | -            | Конечное значение (план)     |
| [`color`](#цвет-и-значения)    | `string`          | -            | Цвет линии                   |
| [`summary`](#цвет-и-значения)  | `string, number`  | -            | Подпись рядом с линией       |
| [`ticks?`](#деления)           | [`Tick[]`](#tick) | -            | Деления                      |
| [`caption?`](#заголовок)       | `React.ReactNode` | -            | Заголовок                    |

### Tick

| Свойство            | Тип              | По умолчанию | Описание                                     |
| ------------------- | ---------------- | ------------ | -------------------------------------------- |
| [`value`](#деления) | `number`         | -            | Значение, у которого нужно поставить деление |
| [`label`](#деления) | `string, number` | -            | Подпись к делению                            |

## Пример

```tsx
import { ProgressBar } from '@consta/widgets/ProgressBar'

<ProgressBar
  data={[
    {
      value: 50,
      valueMin: 0,
      valueMax: 100,
      summary: '50 штук',
      color: 'var(--color-bg-warning)',
    },
  ]}
  size="s"
/>
```
